# PEC 2 - Técnicas de Visualización de Datos

## 📋 Información del Proyecto

Este proyecto implementa tres técnicas de visualización de datos asignadas para la PEC 2 de la UOC:

1. **INFOGRAFÍA** (Grupo I - Técnicas Básicas y Populares)
2. **GAUGE DIAGRAM** (Grupo II - Técnicas Habituales y Específicas)
3. **SPIRAL PLOT** (Grupo III - Técnicas Menos Habituales o Específicas)
